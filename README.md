# Day-Vibe-Cinema

Eng:
================================================================================================================================================================================================
This project allows you to get a movie recommendation based on what day it is today and what the weather is like outside.
To run this project, you must install the libraries specified in the requirements.txt file. After running the main.py file, the program will also check for the presence of the IMDb database. If it is unavailable, you will be prompted to download it using the official IMDb tool. Without installing the database, further steps are either impossible or pointless.
Internet access is required for proper operation to retrieve weather information. Your data will never be removed from your device's use for weather data requests, which is why you specify your city during registration. You can verify this with check the code.
main.py start only from Data.
Some of the code was written by AI.

Ru:
================================================================================================================================================================================================
Этот проект позволяет получать рекомендации по фильмам в зависимости от того, какой сегодня день и какая погода за окном.
Для запуска данного проекта нужно установить библиотеки указанные в requirements.txt. Также после запуска файла main.py программа проверит наличие открытой базы данных IMDb. Если её нет, вам будет предложено скачать её при помощи официального инструмента IMDb. Без установки базы данных дальнейшие действия невозможны и или не имеют смысла.
Для коректной работы требуется доступ в интернет для получения сведений о погоде. Ваши данные не при каких обстоятельсвах не могут покинуть ваше устройство, за исключением запроса данных о погоде, для чего вы и указываете город при регистрации. Вы можете проверить код и убедиться в этом сами.
Файл main.py запускаеть только из папки Data.
Часть кода была написана искусственным интеллектом.
